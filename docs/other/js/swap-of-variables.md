# 交换两个变量的值

## 没难度的解法

### 很明显不给分版本

```javascript
let x = 1, y = 2;
let temp
temp = x
x = y
y = temp
```

非常传统容易理解适合初学者学习的一种思维方式

### 可能给分的ES6解构版

```javascript
let x = 1, y = 2;
[ x, y ] = [ y, x ]

```

主要考察的是对 ES6 的语法 api 熟悉程度（听没听过的那种程度）没啥好说的。

## 有难度的级别

### 很大可能给分的异或版（需要整数）

```javascript
let x = 1, y = 2;
x = x ^ y
y = x ^ y
x = x ^ y

/*
二进制中 需要知道
 0^0 = 0 
 1^1 = 0  
 1^0 = 0^1 = 1
 
* x = 00000001
* y = 00000010
* x = x ^ y = 00000011   |    y = 00000010
* y = x ^ y = 00000001   |    x = 00000011
* x = x ^ y = 00000010   |    y = 00000001  x = 00000010 交换完成
* */

```

### 蛮大可能给分的加减法版与乘除法版

```javascript
let x = 1, y = 2;
x = x + y
y = x - y
x = x - y
```

```javascript
let x = 1, y = 2;
x = x * y
y = x / y
x = x / y
```
